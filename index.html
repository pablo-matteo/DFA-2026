Preguntas frecuentes

¿Es obligatorio que el ciudadano cree una cuenta de usuario para realizar el trámite?

No es obligatorio. Los operadores pueden realizar un registro asistido para apoyar al ciudadano. Sin embargo, se recomienda que el ciudadano cree su propia cuenta para facilitar el seguimiento de sus trámites.

¿En cuánto tiempo máximo debe resolverse una solicitud?

El tiempo óptimo de atención del trámite es de 1 a 2 días hábiles. No obstante, el plazo máximo establecido para emitir una resolución es de hasta 5 días hábiles.

¿Cómo se pueden agregar nuevas carreras o grados?

La incorporación de nuevas carreras o grados debe ser solicitada formalmente a la Dirección de Formación Artística.

¿Cómo se puede solicitar el acceso de nuevos operadores para un centro educativo?

La Dirección de Formación Artística debe remitir un oficio a la Dirección de Sistemas Informáticos indicando: nombre completo del operador, número de DPI, trámite al que solicita acceso, correo electrónico con el que se creó la cuenta, centro educativo correspondiente y la especificación de que se requiere el rol de operador en la plataforma Ventanilla Ágil.
