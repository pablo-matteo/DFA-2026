Preguntas frecuentes

¿Es obligatorio que el ciudadano cree una cuenta de usuario para realizar el trámite?

No es obligatorio. Los operadores pueden realizar un registro asistido; sin embargo, se recomienda que el ciudadano cree su propia cuenta para facilitar el seguimiento.

¿En cuánto tiempo máximo debe resolverse una solicitud?

El plazo máximo establecido para emitir una resolución es de hasta 5 días hábiles.

¿Cómo se pueden agregar nuevas carreras o grados?

Debe solicitarse formalmente a la Dirección de Formación Artística.

¿Cómo se solicita acceso para nuevos operadores?

Mediante oficio dirigido a la Dirección de Sistemas Informáticos, con los datos completos del operador y el centro educativo correspondiente.
